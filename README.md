## O que você precisa

- Node.js 22.15.0

## Instalar dependências

```bash
npm install
# ou
yarn install
```

## Iniciando

Primeiro, execute o servidor de desenvolvimento:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
